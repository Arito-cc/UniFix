# UniFix – Full Stack Issue Reporting System

UniFix is a full-stack MERN application for reporting and managing campus issues with role-based access, authentication, and real-time tracking.

## 🚀 Features
- Role-based dashboards for students and staff  
- Submit, track, and resolve issues  
- JWT-based authentication and secure user management  
- Upvote system to prioritize important issues  
- Department-based workflow and tracking  

---

## 🛠 Tech Stack
- Frontend: React, Tailwind CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT, Bcrypt.js  

---

## ⚙️ Setup
1. Clone the repository  
2. Install dependencies in client and server  
3. Configure `.env` variables  
4. Run frontend and backend
